# Add Cmd and PowerShell command with admin rule to right menu
The reg script is referenced from: [https://blog.csdn.net/cxrsdn/article/details/84538767]  

## Right Menu
* 在此处打开命令行窗口
* 在此处打开命令行窗口(管理员)

## Shift + Right Menu
* 在此处打开命令行窗口
* 在此处打开 Powershell 窗口
* 在此处打开 Powershell 窗口(管理员)
* 在此处打开命令行窗口(管理员)
